---
name: Mobile is Capacitor
description: Mobile app is built in React empacotado com Capacitor, shares same Supabase backend with Web and Desktop
type: feature
---
O app mobile é construído em React empacotado com Capacitor (não React PWA).
Telas: auth, home, games (detail, comments), marketplace (chat), notifications, profile, settings.
Services: game_service, supabase_service, theme_service.
Compartilha o mesmo banco de dados Supabase que Web e Desktop.
