# Web · Termos de Uso (/termos)

## Propósito
Documento legal estático.

## Achados P1
- Versão do termo não é registrada; se mudar, usuários existentes não são notificados nem forçados a re-aceitar. Criar `site_settings.termos_versao` + `profiles.termos_aceite_versao`.
- Sem índice/sumário navegável.
- Sem data de última atualização visível no topo.
